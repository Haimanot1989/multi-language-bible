# Multi-Language Bible

Read Bible scripture in four languages from one place:
- 🇳🇴 Norwegian (Bokmål)
- 🇬🇧 English (KJV)
- 🇪🇷 Tigrinya
- 🇪🇹 Amharic

## Features
- Search any verse by reference (e.g., "Psalms 2:7", "John 3:16")
- View all 4 translations side by side
- One-click copy formatted for Notion

## Tech Stack
- [Astro](https://astro.build) + React islands
- Tailwind CSS
- Self-contained Bible data (JSON)

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

```bash
# Scrape Bible data from sources
npm run scrape

# Build for production
npm run build
```
